# ¿Cómo empezar a estudiar?

Este curso introductorio al desarrollo de páginas web cuenta con una modalidad de aprendizaje en el que se recomienda que el estudiante tome notas y practique a sus tiempos.

Este material se mantendrá en línea 24/7 y solo se comunicará en el caso de actualización.

La plataforma esta en fase de prueba cualquier error encontrado será contemplado

## Software para aprender

Se recomienda que el estudiante primero tenga conocimiento de como instalar programas en su computadora (u ordenador) personal. Por lo general en computadoras ajenas no se recomienda dejar información que sea de utilidad para el estudiante.

### Visual Studio Code

[Link para descargar](https://code.visualstudio.com/download)
En primer lugar y de gran utilidad esta el editor de código Visual Studio Code. No confundir con el [IDE Visual Studio](https://visualstudio.microsoft.com/es/)

Es un software diseñado para multiples lenguajes de programación que nos permite escribir y potenciar el desarrollo de programación independiente del lenguaje (nos servirá para el curso y nos podra servir para todo el curso futuro)

## Atajos para recordar y usar

| Combinacion de teclas                        | Accion                                                        |
| -------------------------------------------- |---------------------------------------------------------------|
| `Ctrol + C`, `Ctrol + V`, `Ctrol + X`        | Copiar / Pegar / Cortar                                       |
| `Ctrol + Z`, `Ctrol + Y`                     | Deshacer / Rehacer                                            |
| `Ctrol + S` o `Ctrol + G`                    | Guardar (en la mayoria de los casos es con **s**)             |
| `Alt + Tab` / `Win + Tab`                    | Cambio de ventana / cambio de escritorio                      |
| `Ctrol + Flecha izq` / `Ctrol + Flecha der`  | Moverse de palabra en palabra izquierda o derecha             |
| `Ctrol + Shift + Flecha izq / flecha der`    | Moverse seleccionando palabra por palabra                     |
| `Inicio` / `Fin`                             | Ubicarse al principio o al final de cada renglón              |
| `Shift + Inicio` / `Shift + Fin`             | Seleccionar el renglón completo                               |
| `Ctrol + Scroll (uso de la rueda del mouse)` | Aumentar / alejar. Se usa para editores y otros recursos $^1$ |
|                                              |                                                               |
